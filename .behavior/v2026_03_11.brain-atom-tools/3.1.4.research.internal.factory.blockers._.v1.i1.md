## feedback loops

**status:** ⚠️ acceptable, not a blocker

**slowest step:** API call latency (~1-3s per call)

each tool use test requires:
1. initial prompt → brain responds with tool_use (~1-2s)
2. continuation with tool_result → brain responds with output (~1-2s)

total per test: ~2-4s minimum

**mitigation already in place:**
- haiku for fast tests [5]
- useThen to share expensive calls [6]

**not a blocker** because test count is small and API latency is unavoidable for integration tests.
