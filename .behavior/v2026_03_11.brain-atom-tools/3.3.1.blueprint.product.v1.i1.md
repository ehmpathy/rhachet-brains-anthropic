# blueprint: product

## summary

extend `genBrainAtom` to support `plugs.tools` for tool use:
- accept `BrainPlugToolDefinition[]` via `plugs.tools`
- translate tool definitions to anthropic format via cast operations
- handle `tool_use` content blocks in response
- enable continuation with `BrainPlugToolExecution[]` as prompt
- return `BrainPlugToolInvocation[]` via `calls.tools`
- extract output and calls independently (not mutually exclusive)

## implementation notes

### cast operation: castIntoAnthropicToolDef

```ts
// src/infra/cast/castIntoAnthropicToolDef.ts
import type Anthropic from '@anthropic-ai/sdk';
import type { BrainPlugToolDefinition } from 'rhachet/brains';
import { asJsonSchema } from '../schema/asJsonSchema';

/**
 * .what = cast rhachet tool definition to anthropic format
 * .why = explicit boundary between rhachet domain and provider SDK
 */
export const castIntoAnthropicToolDef = (input: {
  definition: BrainPlugToolDefinition;
}): Anthropic.Messages.Tool => ({
  name: input.definition.slug,
  description: input.definition.description,
  input_schema: asJsonSchema({
    schema: input.definition.schema.input,
  }) as Anthropic.Messages.Tool['input_schema'],
});
```

### cast operation: castIntoAnthropicToolResult

```ts
// src/infra/cast/castIntoAnthropicToolResult.ts
import type { BrainPlugToolExecution } from 'rhachet/brains';

/**
 * .what = cast rhachet tool execution to anthropic tool_result block
 * .why = explicit boundary between rhachet domain and provider SDK
 */
export const castIntoAnthropicToolResult = (input: {
  execution: BrainPlugToolExecution;
}): { type: 'tool_result'; tool_use_id: string; content: string; is_error: boolean } => ({
  type: 'tool_result' as const,
  tool_use_id: input.execution.exid,
  content: JSON.stringify(input.execution.output),
  is_error: input.execution.signal !== 'success',
});
```

### cast operation: castFromAnthropicToolUse

```ts
// src/infra/cast/castFromAnthropicToolUse.ts
import type Anthropic from '@anthropic-ai/sdk';
import type { BrainPlugToolInvocation } from 'rhachet/brains';

/**
 * .what = cast anthropic tool_use block to rhachet invocation
 * .why = explicit boundary between provider SDK and rhachet domain
 */
export const castFromAnthropicToolUse = (input: {
  block: Anthropic.Messages.ToolUseBlock;
}): BrainPlugToolInvocation => ({
  exid: input.block.id,
  slug: input.block.name,
  input: input.block.input,
});
```

### independent extraction pattern

```ts
// extract output and calls INDEPENDENTLY (not mutually exclusive)
// model can return both text AND tool_use blocks simultaneously

// extract tool calls (always, regardless of text presence)
const toolUseBlocks = response.content.filter(
  (block): block is Anthropic.Messages.ToolUseBlock => block.type === 'tool_use',
);
const calls = toolUseBlocks.length > 0
  ? { tools: toolUseBlocks.map((block) => castFromAnthropicToolUse({ block })) }
  : null;

// extract text block (always, regardless of tool_use presence)
const textBlock = response.content.find(
  (block): block is Anthropic.TextBlock => block.type === 'text',
);

// try to parse output from text, with graceful fallback
const output = (() => {
  if (!textBlock?.text) return null;
  try {
    return askInput.schema.output.parse(JSON.parse(textBlock.text));
  } catch {
    // parse failed — if tool calls present, this is expected (partial response)
    // if no tool calls, this is unexpected — let it throw on next attempt
    if (calls) return null;
    throw new Error('structured output parse failed with no tool calls');
  }
})();

// both output AND calls can be present simultaneously
// e.g., "Let me look that up for you" + tool_use block
```

### tool exchanges use standard exchange pattern

```ts
// tool exchanges serialize the same as message exchanges
// no special-case code needed

const exchangeInput = isToolResultContinuation
  ? JSON.stringify(currentPromptContent)  // tool_result blocks as JSON
  : (askInput.prompt as string);           // plain string

const exchangeOutput = toolUseBlocks.length > 0
  ? JSON.stringify(response.content)       // full content with tool_use blocks
  : outputText;                            // plain structured output
```
