# playtest: brain atom tools

## prerequisites

- node 18+
- ANTHROPIC_API_KEY environment variable set
- pnpm installed
- repo built: `npm run build`

## setup

```bash
# ensure dependencies installed
pnpm install --frozen-lockfile

# build the package
npm run build

# run the integration tests directly
source .agent/repo=.this/role=any/skills/use.apikeys.sh && npm run test:integration -- genBrainAtom
EOF
```

## happy paths

### 1. tool invocation

**action:** run the integration tests

**expected outcome:**
- case5: output null, calls.tools defined, invocation has exid/slug/input
- case7: continuation with tool result produces final output
- case8: brain uses the tool output value

### 2. direct answer without tools

**expected outcome:**
- case6: output defined, calls null (brain answered directly)

### 3. parallel tool calls

**expected outcome:**
- case11: calls.tools contains multiple invocations with unique exids

## edgey paths

### error signal handle

**expected outcome:**
- case9: brain reasons about constraint violation (error:constraint)
- case10: brain reasons about malfunction (error:malfunction)

### tools + structured output

**expected outcome:**
- case12: output conforms to schema, result is correct

## pass/fail criteria

- **pass if:**
  - all 45 tests pass
  - no errors thrown

- **fail if:**
  - any test fails
  - tool invocations lack exid, slug, or input
  - continuation fails to produce output
  - parallel calls have duplicate exids
  - metrics not computed
