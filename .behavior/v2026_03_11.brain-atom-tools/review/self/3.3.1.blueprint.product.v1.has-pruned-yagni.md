## "w/o output schema" question

**wish says:** "w/ and w/o output schema"

**implementation:** always uses structured outputs (output_format required by betaZodOutputFormat).

**question:** did we miss "w/o output schema" test?

**analysis:** rhachet's `BrainAtom.ask()` requires `schema: { output }`. the interface does not support schema-less calls. the wish phrase "w/ and w/o output schema" likely means:
- WITH schema: brain provides structured answer (case12)
- WITHOUT schema being used yet: brain calls tools, output is null (case5)

**verdict:** interpretation aligns with rhachet contract. no yagni.
