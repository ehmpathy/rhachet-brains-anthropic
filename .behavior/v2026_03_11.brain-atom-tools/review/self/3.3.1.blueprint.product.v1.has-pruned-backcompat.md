# review: has-pruned-backcompat

## reflection

examined each backwards compatibility measure in the implementation. asked: was this explicitly requested or added "to be safe"?

---

## backwards compatibility measures reviewed

### 1. TPlugs default type parameter

**code:** `TPlugs extends BrainPlugs = BrainPlugs`

**question:** was this explicitly requested?

**analysis:** yes. rhachet's contract prescribes the type signature. the default value `= BrainPlugs` is standard typescript pattern for optional generics. this enables callers to omit the type parameter when not passing tools.

**verdict:** prescribed by rhachet contract. not added "to be safe."

---

### 2. optional plugs parameter

**code:** `plugs?: TPlugs`

**question:** was this explicitly requested?

**analysis:** yes. the wish says "support `{ plugs: { tools } }`" — the `plugs` is optional by design. callers without tools should not need to pass an empty object.

**verdict:** prescribed by wish. not added "to be safe."

---

### 3. AsBrainPromptFor conditional type

**code:** `prompt: AsBrainPromptFor<TPlugs>`

**question:** does this add backwards compat we didn't need?

**analysis:** `AsBrainPromptFor<TPlugs>` resolves to `string` when no tools are plugged, and `string | BrainPlugToolExecution[]` when tools are plugged. this is rhachet's prescribed type, not a backwards compat shim.

**verdict:** prescribed by rhachet contract.

---

### 4. genBrainRepl.ts fail-fast checks

**code:**
```ts
// fail-fast: plugs.tools is not supported for repls
if (input.plugs?.tools && input.plugs.tools.length > 0) {
  throw new BadRequestError(...);
}

// fail-fast: tool result continuation is not supported
if (Array.isArray(input.prompt)) {
  throw new BadRequestError(...);
}
```

**question:** are these backwards compat shims?

**analysis:** no. these are explicit rejections of unsupported features, not backwards compatibility measures. they prevent callers from calling operations the repl cannot support. this is correct fail-fast behavior.

**verdict:** fail-fast guards, not backwards compat shims.
