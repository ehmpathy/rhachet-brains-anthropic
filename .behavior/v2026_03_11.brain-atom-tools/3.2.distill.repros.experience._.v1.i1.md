# distill: experience reproductions

## experience reproductions

based on vision and criteria, here are the user experiences to reproduce:

| experience | entry point | user actions | expected outcome | test type |
|------------|-------------|--------------|------------------|-----------|
| tool invocation | `atom.ask({ plugs: { tools } })` | prompt that needs tools | `output: null`, `calls.tools` has invocations | integration |
| direct answer | `atom.ask({ plugs: { tools } })` | prompt that doesn't need tools | `output` defined, `calls: null` | integration |
| tool result continuation | `atom.ask({ on: { episode }, prompt: executions })` | continue with tool results | brain uses results, `output` defined | integration |
| success signal | continue with `signal: 'success'` | execute tool, return success | brain uses output value | integration |
| error:constraint signal | continue with `signal: 'error:constraint'` | tool fails with bad input | brain reasons about violation | integration |
| error:malfunction signal | continue with `signal: 'error:malfunction'` | tool has internal error | brain reasons about failure | integration |
| parallel tool calls | prompt needs multiple tools | brain requests multiple tools | `calls.tools` has N invocations, unique exids | integration |
| tools + output schema | ask with tools and schema | brain produces final answer | output conforms to schema | integration |
| tools without schema | ask with tools, no schema | brain produces final answer | output is unstructured text | integration |
| episode tracked | complete tool workflow | workflow completes | episode.exid returned, metrics computed | integration |

## reproduction feasibility

### available utilities

- `test-fns`: given/when/then, useThen, getError
- `genBrainAtom`: create atoms with slug
- `genBrainPlugToolDeclaration`: create test tools (from rhachet)
- `asBrainPlugToolDict`: O(1) lookup (from rhachet)
- zod: schema definitions

### setup required

1. define test tool(s) with known behavior
2. create atom instance (haiku for speed, sonnet for continuation)
3. craft prompts that trigger tool use

### concrete test sketch

```ts
given('[case5] tool invocation', () => {
  // define calculator tool
  const calculatorTool = genBrainPlugToolDeclaration({
    slug: 'test.calculator',
    name: 'Calculator',
    description: 'performs arithmetic: add, subtract, multiply, divide',
    schema: {
      input: z.object({ a: z.number(), b: z.number(), op: z.string() }),
      output: z.object({ result: z.number() }),
    },
    execute: async ({ invocation }) => {
      const { a, b, op } = invocation.input;
      const result = op === '+' ? a + b : op === '-' ? a - b : op === '*' ? a * b : a / b;
      return { result };
    },
  });

  const tools = [calculatorTool];
  const toolDict = asBrainPlugToolDict(tools);

  when('[t0] prompt requires tool use', () => {
    const result = useThen('brain requests tool', async () =>
      brainAtomHaiku.ask({
        role: {},
        prompt: 'what is 7 * 8? use the calculator tool',
        schema: { output: z.object({ answer: z.number() }) },
        plugs: { tools },
      }),
    );

    then('output is null', () => {
      expect(result.output).toBeNull();
    });

    then('calls.tools contains invocation', () => {
      expect(result.calls).toBeDefined();
      expect(result.calls?.tools).toHaveLength(1);
      expect(result.calls?.tools[0].slug).toEqual('test.calculator');
    });

    then('invocation has exid', () => {
      expect(result.calls?.tools[0].exid).toBeDefined();
    });
  });

  when('[t1] continued with tool result', () => {
    const resultFirst = useThen('brain requests tool', async () =>
      brainAtomSonnet.ask({
        role: {},
        prompt: 'what is 7 * 8? use the calculator',
        schema: { output: z.object({ answer: z.number() }) },
        plugs: { tools },
      }),
    );

    const resultSecond = useThen('continuation succeeds', async () => {
      const executions = await Promise.all(
        resultFirst.calls!.tools.map(async (inv) => {
          const tool = toolDict[inv.slug];
          return tool.execute({ invocation: inv }, {});
        }),
      );
      return brainAtomSonnet.ask({
        on: { episode: resultFirst.episode },
        role: {},
        prompt: executions,
        schema: { output: z.object({ answer: z.number() }) },
        plugs: { tools },
      });
    });

    then('output is defined', () => {
      expect(resultSecond.output).toBeDefined();
      expect(resultSecond.output?.answer).toEqual(56);
    });
  });
});
```
